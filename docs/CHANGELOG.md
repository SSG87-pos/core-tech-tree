# Changelog

## 2026-06-11

- `Core Technology 2026` 단일 HTML 화면을 Level 1 → Level 2 → Core 기술 구조로 재정리했다.
- 필터를 중복 선택 방식으로 바꾸고, 필터 접힘 상태에서도 `기술 / 조직` 관점 전환을 유지하도록 조정했다.
- 기술 체계 맵을 더 조밀한 트리형 카드 구조로 정리하고, Level 1 Core 개수 pill, Level 2 개수 표시, 성장 단계별 별 색상, 태그/조직/담당자 pill을 재정리했다.
- 조직 관점은 연구그룹과 팀을 별도 섹션으로 나누고, 포함 기술이 많은 조직이 위로 오도록 정렬했다.
- 상세 팝오버에서 기술명, 성장 단계, 연구단계, 태그, 요약, 수행 조직, 담당자를 수정할 수 있게 했다.
- 상세 팝오버에서 기술 삭제와 삭제 되돌리기를 지원한다.
- `구조 편집` 모드에서 기술 추가, 기술 카드의 Level 2 이동, Level 2 블록의 Level 1 이동, 이동 되돌리기를 지원한다.
- Level 1 분포는 도넛 차트, Level 2 분포는 Level 1별 막대 그룹으로 바꾸고, 성장 단계/연구단계 축은 데이터가 비어도 기준 항목을 유지한다.
- XLSX 템플릿과 JSON schema를 `schemaVersion: 3` 기준의 한 시트 Core 기술 데이터 구조로 정리했다.
- NHN Cloud Enterprise 계열의 밝고 정돈된 톤으로 색상, 굵기, pill, 카드 경계, 조직 관점 타이포 위계를 조정했다.
